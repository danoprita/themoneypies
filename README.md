Build and run
-------------
 - Clone repository
 - `$ cd themoneypies`
 - Install npm (NodeJS)
 - `bower install`
 - `mvn package`
 - `cp target/themoneypies-1.0-SNAPSHOT.jar another-directory`
 - `cd another-directory`
 - `java -jar themoneypies-1.0-SNAPSHOT.jar`
 - Browse to [http://localhost:8081/index.html](http://localhost:8081/index.html)
